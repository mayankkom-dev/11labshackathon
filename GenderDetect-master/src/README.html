LIUM_SpkDiarization is a software package dedicated to speaker diarization (ie speaker segmentation and clustering). Most of the tools in the package take a segmentation as input and generate a new segmentation as output. The provided tools allow to perform BIC hierarchical clustering, Viberbi decoding using GMM models trained by EM or MAP, and CLR or NCLR hierarchical clustering using GMM (clustering based over automatic speaker recognition methods).
